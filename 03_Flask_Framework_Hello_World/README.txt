1. Create a virtual environment (optional but recommended):
   python -m venv venv

   (Linux/macOS) source venv/bin/activate

3. Install Flask:
   pip install Flask

4. Run the app:
   (Linux/macOS)
   export FLASK_APP=hello_app.py
   flask run

   (Windows CMD)
   set FLASK_APP=hello_app.py
   flask run
